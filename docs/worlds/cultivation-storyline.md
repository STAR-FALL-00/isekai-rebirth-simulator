## 1. 世界观概述

### 1.1 境界体系

修仙界采用经典的八境界体系，每突破一境，寿元大幅增长：

| 阶段 | 境界名称 | 基础寿元 | 特征 |
|:----:|:--------:|:--------:|:-----|
| 0 | 炼气期 | 100岁 | 修仙起点，引气入体 |
| 1 | 筑基期 | 180岁 | 凡体化灵体，踏上长生路 |
| 2 | 金丹期 | 300岁 | 凝结金丹，法力浑厚数倍 |
| 3 | 元婴期 | 500岁 | 元婴出世，相当于第二条命 |
| 4 | 化神期 | 800岁 | 感应天地法则，引动异象 |
| 5 | 渡劫期 | 1200岁 | 直面天劫，半步仙人 |
| 6 | 大乘期 | 2000岁 | 触摸仙界门槛，静待飞升 |
| 7 | 飞升 | 9999岁 | 破碎虚空，羽化登仙 |

### 1.2 核心属性含义

| 属性 | 中文名 | 含义 |
|:----:|:------:|:-----|
| strength | 体质 | 肉身强度，影响战斗与承受力 |
| intelligence | 悟性 | 领悟功法、突破瓶颈的能力 |
| charisma | 机缘 | 人际交往、获取机遇的能力 |
| luck | 气运 | 奇遇概率、逢凶化吉的运气 |
| health | 气血 | 生命值，归零即死亡 |
| special | 灵根 | 修仙核心资质，决定修炼上限 |

### 1.3 主题定位

修仙界以"踏碎虚空，追求长生"为核心主题。玩家从微末起步，历经渡劫飞升，探索上古遗迹，与诸天万族争锋。故事围绕**长生与红尘的冲突**、**正邪的模糊边界**、**天赋与毅力的对抗**三大核心矛盾展开。

## 2. 身份专属故事线（10个身份）

每个身份拥有**8个链式专属事件**：童年×2 → 成长×2 → 成年×2（含选择分支） → 特殊×2。事件通过 `chainPriority` 和 `requiredFlags` 形成严格的因果链条。

---

### 2.1 凡人子弟（Commoner）

**背景设定**：出身平凡村落，被判定为五行杂灵根/下品之资，修仙无望。

**童年事件链**：
1. **灵根检测失败**（3-8岁）：在青牛村被判定为下品灵根，村人目光从期待变怜悯。但你发现了一处被藤蔓遮掩的神秘石洞。
2. **不甘心的坚持**（7-12岁）：每日清晨爬最高处吐纳，或发现石洞刻着古怪符文。你不信命。

**成长事件链**：
1. **体修启蒙**（14-20岁）：在石洞中找到《金刚淬体诀》，以妖兽之血淬体。十六岁以肉身击杀低阶妖兽，震惊众人。
2. **初获认可**（19-25岁）：救青云宗外门弟子获赠筑基丹，或遇到看出你体修之路的前辈。

**成年事件链**：
1. **破格入门**（26-35岁）：以体修之身通过宗门选拔，成为青云宗史上第一个体修外门弟子，却遭遇正统修士轻视。
2. **道路抉择**（36-45岁）：
   - **坚持体修**（成功率60%）：踏上不归路，以力证道
   - **转修正统**（成功率75%）：放弃体修，获得宗门资源但失去特色

**特殊事件链**：
1. **混沌灵根觉醒**（10-30岁）：生死危机中肉身突破极限，发现灵根并非没有，而是被九重封印锁住的**混沌灵根**！
2. **体灵双修**（30-60岁）：解开第一重封印，混沌灵根与金刚肉身完美融合，开创前所未有的"体灵双修"之道，被后人称为"混沌金刚"。

**起承转合**：从被判定为废材的绝望，到以体修另辟蹊径的坚持，再到混沌灵根觉醒的惊天逆转，最终成为体灵双修的传奇——一个关于"起点不能决定终点"的完美诠释。

### 2.2 宗门天才（Genius）

**背景设定**：天生灵根卓越，被青云宗玄天剑尊收为关门弟子，起点极高。

**童年事件链**：
1. **出生异象**（3-8岁）：紫气东来三千里，三岁引气入体，五岁御剑飞行。既有羡慕也有嫉妒的目光。
2. **高压童年**（7-12岁）：六岁宗门大比以炼气击败筑基，被誉为"千年第一人"，但每日挥剑三千次，掌心磨出血泡。

**成长事件链**：
1. **卷入派系**（14-20岁）：十四岁突破筑基创纪录，但发现玄天剑尊年事已高，长老争夺掌门之位，大师兄拉拢你站队。
2. **战胜心魔**（19-25岁）：秘境试炼中遭遇心魔——最恐惧被更天才的人超越。战胜后道心坚定，明白"修仙修的不是天赋"。

**成年事件链**：
1. **正邪交锋**（26-35岁）：成为青云宗最年轻金丹修士，发现大师兄暗中与魔道交易。
2. **抉择时刻**（36-45岁）：
   - **举报大师兄**（成功率55%）：可能被反诬，风险极高
   - **隐忍收集证据**（成功率75%）：暗中掌握完整证据链

**特殊事件链**：
1. **掌门之位**（10-30岁）：玄天剑尊出关处决大师兄，将掌门信物交给你——你拒绝，说出"我的道不在宗门之内"。
2. **开创新宗**（30-60岁）：创立"新青云宗"，不分出身、不论灵根，有教无类。百年后成为修仙界最年轻化神修士。

**起承转合**：天才光环下的孤独与压力 → 卷入宗门权力漩涡 → 以道心战胜心魔 → 拒绝掌门、走出自己的道路 → 创立有教无类的新宗门。天才的终极觉醒：真正的强大不是天赋，是选择。

### 2.3 散修野客（Wanderer）

**背景设定**：无门无派，街头乞讨长大，独来独往，适应力极强。

**童年事件链**：
1. **街头求生**（3-8岁）：最早记忆是在街头乞讨，老乞丐教你如何在妖兽追赶时活下来。
2. **捡到玉简**（7-12岁）：垃圾堆里捡到半块破损玉简，被认出是上古散修"逍遥子"的遗物。

**成长事件链**：
1. **洞府传承**（14-20岁）：进入废弃洞府，获得《逍遥剑诀》和断剑。与散修强盗搏斗保住性命。
2. **领袖萌芽**（19-25岁）：在散修集市结识盟友，得知散修联盟酝酿起义打破宗门垄断。开始帮助其他散修，名声渐起。

**成年事件链**：
1. **领袖担当**（26-35岁）：成为散修领袖，青云宗招安、万法宗威胁剿灭。你在集会上演讲："我们没有资源，没有庇护，但我们有自由！"
2. **最后通牒**（36-45岁）：
   - **领导起义**（成功率50%）：以武力打破宗门垄断
   - **谈判妥协**（成功率75%）：争取有限自主权

**特殊事件链**：
1. **逍遥子后裔**（10-30岁）：起义前夕得知自己是逍遥子直系后裔，玉简中藏着完整传承。上古散修曾与宗门平起平坐。
2. **散修之城**（30-60岁）：攻下领地建立第一个"散修城池"，不分出身、有教无类。百年后成为修仙界第五大势力，人称"逍遥散仙"。

**起承转合**：孤儿乞讨的底层开局 → 捡到上古传承的奇遇 → 从独行侠到散修领袖的蜕变 → 身份揭秘的血脉觉醒 → 建立自由之城、改变修仙格局。主题是"自由高于一切"。

### 2.4 妖族血脉（Demon Blood）

**背景设定**：体内流淌上古妖族血脉，龙族与虎族混血，月圆之夜瞳孔变金。

**童年事件链**：
1. **怪物之名**（3-8岁）：瞳孔月圆变金、牙齿尖锐，被村人叫"怪物"。一次玩耍手臂长出鳞片，村民拿起农具。
2. **母亲离去**（7-12岁）：母亲留下玉佩离去，被告知来自万妖森。被猎人追赶时血脉第一次觉醒，逃入深山。

**成长事件链**：
1. **血脉控制**（14-20岁）：神秘人看出你是龙族与虎族混血，万兽帝君一直在找你。开始在瀑布下修炼控制妖血。
2. **妖王祭典**（19-25岁）：万兽帝君送来请帖，但养父恳求不要与妖族往来。妖族少女告诉你："在人类世界你是怪物，在妖族世界你是天才。"

**成年事件链**：
1. **身份危机**（26-35岁）：正道宗门发现妖族血脉要将你逐出，妖族希望你完全化妖。遇到混血家庭："我们不是人也不是妖，我们只是我们自己。"
2. **化妖抉择**（36-45岁）：
   - **彻底化妖**（成功率50%）：成为妖族大将军
   - **保持人形**（成功率75%）：永远不得再入万妖森

**特殊事件链**：
1. **母亲下落**（10-30岁）：发现母亲被青云宗秘密囚禁千年，要用龙族血脉炼制"化龙丹"。与玄天剑尊对峙。
2. **龙虎混元**（30-60岁）：救出母亲后身负重伤，龙族与虎族血脉完美融合，成为前所未有的"龙虎混元体"。成为人妖两族桥梁，促成和平协议。

**起承转合**：被当作怪物的童年创伤 → 血脉觉醒的力量与恐惧 → 人妖两族的身份撕裂 → 母亲被囚的惊天阴谋 → 血脉融合、成为和平桥梁。主题是"身份不是标签，选择才是"。

### 2.5 魔道遗孤（Demon Heritage）

**背景设定**：魔道覆灭后的遗孤，身怀禁忌功法《血神经》的秘密。

**童年事件链**：
1. **火海记忆**（3-8岁）：最早记忆是黑衣人屠杀全家，躲在衣柜里不敢出声。被正道修士收养，他说"你父母是好人，只是走错了路"。
2. **发现真相**（7-12岁）：在养父书房发现被封印古籍，看到父亲名字——"血魔殿左护法"。养父告知真相：父母从未杀过一个凡人。

**成长事件链**：
1. **魔血觉醒**（14-20岁）：遭遇危险时魔道血脉第一次觉醒，黑暗力量反杀敌人。开始修炼《血神经》，发誓只杀该死之人。
2. **正魔之问**（19-25岁）：血魔殿称你为"少主"，但青云宗弟子不知你身份却与你成为朋友。你开始怀疑：正魔之分真的那么绝对吗？

**成年事件链**：
1. **身份暴露**（26-35岁）：青云宗要处死你，血魔殿要带你回去。站在悬崖边两面都是深渊。正道朋友为你求情却被处罚。
2. **正魔抉择**（36-45岁）：
   - **加入魔道**（成功率50%）：血魔老祖帮你摧毁虚伪正义
   - **坚守正道**（成功率75%）：在正道中走自己的路

**特殊事件链**：
1. **灭门真相**（10-30岁）：调查发现灭门凶手不是正道，而是血魔殿内部嫉妒父亲地位的人！血魔老祖包庇凶手。你独自杀入血魔殿复仇。
2. **中立庇护所**（30-60岁）：不加入任何一方，建立"中立庇护所"收留正魔流亡者："正义和邪恶都是标签，我只问本心。"后人称你为"魔道叛徒"、"正道异端"。

**起承转合**：灭门惨案的童年阴影 → 被正道收养的身份矛盾 → 魔血觉醒的力量诱惑 → 正魔两道的撕裂抉择 → 发现灭门真相的复仇 → 建立超越正邪的庇护所。主题是"超越标签，只问本心"。

### 2.6 转世仙人（Reincarnated）

**背景设定**：前世为仙界大能转世，拥有模糊的前世记忆碎片。

**童年事件链**：
1. **金色梦境**（3-8岁）：从小做同一个梦——金色宫殿，声音说"回来吧"。出生时有仙光护体。三岁吟诵上古仙语。
2. **记忆壁垒**（7-12岁）：六岁头痛欲裂，看到战场画面——一个与自己面貌相同的人正在与天道对抗。识海中有记忆壁垒，有人在阻止你记起前世。

**成长事件链**：
1. **前世追随者**（14-20岁）：黑衣人跪地痛哭："主人，三千年了！"开始觉醒前世能力，不需学习就知结丹凝婴，但被因果缠身。
2. **宿敌降临**（19-25岁）：前世宿敌——仙界大能传人来讨仙器。天机阁暗中保护你，因为每个转世仙人都可能是打破"飞升封印"的关键。

**成年事件链**：
1. **前世之死**（26-35岁）：突破元婴后发现前世并非自然死亡，而是被仙界大能暗杀！前世自己在梦中说："不要追寻过去，活出你自己。"但你已无法回头。
2. **记忆抉择**（36-45岁）：
   - **觉醒前世**（成功率45%）：获得仙人之力但失去今生自我
   - **封印记忆**（成功率75%）：做普通元婴修士

**特殊事件链**：
1. **仙界阴谋**（10-30岁）：觉醒记忆——前世是"逍遥仙君"，因发现仙界阴谋（阻止凡人飞升、独占资源）而被暗杀。神识达仙人层次，看到凡界与仙界间的封印。
2. **向仙界宣战**（30-60岁）：以凡人之躯向仙界宣战，留下话语："我不是为自己报仇，是为所有被仙界欺骗的凡人讨公道。"化作金光冲向天际。

**起承转合**：神秘梦境的童年困惑 → 记忆觉醒的冲击 → 前世因果的纠缠 → 发现被暗杀真相 → 觉醒与自我的抉择 → 为凡人向仙界宣战。主题是"觉醒不是回归过去，是为未来而战"。

### 2.7 先天灵体（Spirit Body）

**背景设定**：天生与灵气极度亲和，不修炼也能自发吸收灵气，但肉身极其脆弱。

**童年事件链**：
1. **灵体天成**（3-8岁）：出生时方圆百里灵气疯狂涌入，接生婆跪地叩拜。不需要修炼自动吸收灵气，但普通摔倒就骨折——"琉璃做的天才"。
2. **雷电共鸣**（7-12岁）：五岁时被雷电击中非但没受伤，反而吸收雷电之力。开始与天地元素共鸣，听到花草树木的"声音"。

**成长事件链**：
1. **肉身危机**（14-20岁）：灵体与灵脉共鸣，修为一日千里，但肉身越来越虚弱——灵气太浓郁，肉体承受不住了。被诊断："灵根可修炼到化神，肉身可能撑不到金丹。"
2. **寻找出路**（19-25岁）：发现"元素化身"之术——放弃肉身化为元素之灵。或遇到器宗长老，提出用通天灵宝重塑根基（成功率不到一成）。

**成年事件链**：
1. **肉身崩溃**（26-35岁）：肉身到达极限，修炼中真的崩溃——灵体漂浮空中，化作一团纯粹灵气。
2. **存亡抉择**（36-45岁）：
   - **元素化**（成功率40%）：成为纯粹元素之灵，获得力量但失去人类身份
   - **重塑肉身**（成功率75%）：冒险重塑，可能成功也可能陨落

**特殊事件链**：
1. **后天灵体**（10-30岁）：重塑肉身成功，新肉身虽不完美但足够坚固，保留元素共鸣。成为第一个"后天重塑的先天灵体"。
2. **元素传说**（30-60岁）：或选择元素化，成为风、成为雷、成为天地间最纯粹的元素。百年后传说：暴风雨来临时会出现人形闪电守护土地。

**起承转合**：天赐灵体的荣耀与诅咒 → 肉身脆弱的生死危机 → 灵体与肉体的终极冲突 → 元素化或重塑的存亡抉择 → 以新形态延续存在。主题是"完美有代价，残缺亦可重生"。

### 2.8 器灵转世（Artifact Spirit）

**背景设定**：上古法宝"斩仙"的器灵转世为人，拥有法宝的部分记忆和神通。

**童年事件链**：
1. **法宝感应**（3-8岁）：对法宝有奇特感应，拿起破铜烂铁会流泪。常梦见自己变成一柄剑被白衣仙人握在手中。被发现魂魄中有器灵印记。
2. **（注意：该身份童年仅1个事件，直接进入成长）**

**成长事件链**：
1. **寻找本体**（14-20岁）：确定本体是上古神剑"斩仙"，失落于葬仙谷。踏上旅途，遇到器宗想回收炼化斩仙剑。
2. **器灵三分**（19-25岁）：找到线索——斩仙剑不在葬仙谷，而在青云宗禁地剑冢中。器灵千年前分裂成三份，你只是其中一份。

**成年事件链**：
