## 一、设计理念：隐晦而非直述

「浮空要塞」的核心设计原则是**通过设定暗示 SAO 的世界观，但绝不直接提及**。

| SAO 原作元素 | 浮空要塞的隐晦表达 | 出现位置 |
|:---|:---|:---|
| 艾恩葛朗特（100层浮空城） | 「悬浮在虚空中的千层巨塔」「传闻到达最顶层的人将获得自由」 | 世界描述、境界名称 |
| 起始之镇 | 「起始之镇」 | 地点词汇库 |
| 安全区 | 「安全区」 | 地点词汇库 |
| 楼层守护者 | 「楼层守护者」「巨像（每25层出现）」 | NPC、战斗事件 |
| 剑技系统（Sword Skills） | 「剑技」属性、境界名称「剑士/骑士/王者」 | 属性名、境界表 |
| 血条/HP | 「生命值」 | 属性名 |
| 桐人（黑衣剑士） | 「暗影」— 居住在湖畔小屋的黑衣剑士 | NPC、事件文本 |
| 亚丝娜（细剑/闪光） | 「晨曦」— 使用细剑的公会干部 | NPC、事件文本 |
| 结衣（AI少女） | 「羽衣」— 在迷宫迷路的失忆少女，隐藏秘密 | NPC、事件文本 |
| 莉兹贝特（锻造师） | 「铁锤」— 热情的锻造师 | NPC、事件文本 |
| POH/红名玩家 | 「血刃」— 以猎杀其他冒险者为乐的PK者 | NPC、事件文本 |
| 克莱因（中二公会会长） | 「角色扮演者」— 沉浸在自己角色中的人 | 身份 |
| 西莉卡（驯兽师） | 「驯兽师」— 与塔中怪物建立羁绊 | 身份 |
| 艾基尔/莉兹（锻造/商人） | 「锻造师」「商人」 | 身份 |
| 系统/管理员 | 「管理员权限卡」（传说道具） | 道具 |
| 不能退出/死亡游戏 | 「跳楼」死亡事件、HP归零即死亡 | 死亡事件 |
| 公会系统 | 「公会领袖」「公会大厅」 | 身份、地点 |
| 攻略组 | 「前线攻略者」 | 身份 |

**刻意避免的词汇**: VRMMO、完全潜行、NERvGear、桐人、亚丝娜、SAO、血条、艾恩葛朗特、茅场晶彦

### 6.2 新增条件评估器

为支持浮空要塞的成就，achievement-engine 新增5个评估器：
- `max_stat_strength` — 力量峰值检测
- `max_realm` — 境界等级检测
- `relationship_level` — 单一NPC羁绊等级
- `relationship_dual` — 双NPC同时达标
- `flag_set` — 特定Flag检测
