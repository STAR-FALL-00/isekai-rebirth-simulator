# 魔法大陆世界观设定集

> **文档定位**：本文档是魔法大陆所有事件、支线、剧情设计的"事实来源"。后续添加任何支线或事件时，必须以本文档的设定为基准，确保逻辑自洽。

## 1. 世界概述

### 1.1 位面结构

魔法大陆并非单一平面世界，而是由**物质界**、**元素位面**、**暗影界**、**星界**四层构成的魔法矩阵。每一层都由不同的魔法法则支配。

| 位面 | 特征 | 进入方式 | 魔力浓度 |
|------|------|----------|----------|
| **物质界** | 凡人生活的主世界，也是玩家初始所在 | 无 | 中等 |
| **元素位面** | 纯粹元素能量的世界，分为火、水、风、土、雷、光、暗七界 | 召唤术/位面传送 | 极高 |
| **暗影界** | 暗影魔法的源头，充满扭曲与诱惑 | 暗影魔法/特殊仪式 | 危险 |
| **星界** | 魔法真理的终极殿堂，连贤者都视为传说 | 未知 | 理论无限 |

> **叙事要点**：物质界是主舞台，元素位面在中后期可通过召唤或传送进入，暗影界是禁忌魔法的来源，星界只存在于传说和隐藏剧情中。

### 1.2 魔力体系

魔力是魔法大陆的根本能量，来源于**元素之核**。物质界的魔力浓度并非均匀分布，而是像洋流一样在天地间流动，形成**魔脉**。

- **魔脉等级**：微型魔脉 → 小型魔脉 → 中型魔脉 → 大型魔脉 → 元素之眼
- **元素潮汐**：每五十年一次元素涨潮，期间施法效率大幅提升；每五十年一次元素退潮，期间魔法效果减弱。
- **元素属性**：火、水、风、土为基础四元素，进阶元素有雷、冰、光、暗。法师的元素亲和力决定施法效率和威力。
- **魔力污染**：禁忌魔法会污染魔力，被污染的魔力若被普通法师吸收，会导致魔力反噬或精神崩溃。

> **数值自洽**：元素亲和（special属性）直接影响施法效率。魔法绝缘者无法感知魔力，只能走奥术/炼金路线。

### 1.3 魔法法则

魔法大陆的运行遵循三大法则：

- **等价交换**：魔法不会凭空创造或毁灭，只能转化。施展强大魔法需要付出相应代价——体力、精神力、甚至寿命。
- **元素相克**：火克风、风克土、土克水、水克火。光与暗互相克制。精通相克的法师可以越级击败强敌。
- **魔力共鸣**：当多位法师同时施法时，魔力会产生共鸣，效果成倍增强。这也是为什么法师公会团战如此强大的原因。
- **真理之眼**：传说中，达到贤者境界的法师可以"看见"魔法的本质——即所有魔法都只是"真理之塔"上的一小块碎片。

## 3. 势力关系图

### 3.1 魔法学院联盟

魔法学院联盟以"传播魔法知识、培养新一代法师"为宗旨，但内部存在严重的派系斗争。

#### 烈焰学院（火系第一学院）
- **院长**：炎之贤者艾格尼斯（大法师巅峰）
- **核心魔法**：《烈焰风暴》《凤凰涅盘术》《熔岩护甲》
- **领地**：烈焰山脉（大型魔脉）
- **当前目标**：独占真理之塔火元素层，压制冰霜学院
- **与其他势力关系**：与冰霜学院**敌对**，与雷霆学院**盟友**，与暗影议会**死敌**
- **特色**：火系攻击力天下第一，但防御力偏弱
- **支线扩展点**：学院内部分为"毁灭派"（主张用火净化一切）和"重生派"（主张火是生命的源泉），玩家可介入派系斗争

#### 冰霜学院（冰系大宗）
- **院长**：霜之女王伊莉雅（大法师巅峰）
- **核心魔法**：《绝对零度》《冰龙召唤》《时间冻结》
- **领地**：永冻高原（大型魔脉）
- **当前目标**：夺回真理之塔冰元素层控制权，报复烈焰学院
- **与其他势力关系**：与烈焰学院**敌对**，与风暴学院**中立**，与暗影议会**敌对**
- **特色**：控制能力最强，但施法速度较慢
- **支线扩展点**：冰霜学院内部有"古典派"（坚守传统冰魔法）和"革新派"（融合水与风的新冰魔法），玩家可选择站队

#### 风暴学院（风雷系圣地）
- **院长**：风暴之主托尔（大法师后期）
- **核心魔法**：《雷霆万钧》《风暴之眼》《闪电链》
- **领地**：风暴群岛（中型魔脉，但元素活跃度极高）
- **当前目标**：保持中立，通过雷电魔法研究打开通往星界的道路
- **与其他势力关系**：与烈焰学院**盟友**，与所有学院**中立偏友好**
- **特色**：速度最快，群攻能力最强
- **支线扩展点**：风暴学院暗中研究"禁雷魔法"——可以劈开空间的雷霆，但成功率极低且极易反噬

#### 大地学院（土系学院）
- **院长**：石心贤者格拉姆（大法师中期）
- **核心魔法**：《大地之盾》《山脉召唤》《化石为泥》
- **领地**：巨石要塞（中型魔脉）
- **当前目标**：守护物质界的稳定，加固元素位面封印
- **与其他势力关系**：与所有势力**中立偏友好**
- **特色**：防御力天下第一，但攻击力偏弱
- **支线扩展点**：大地学院发现了"地下世界"——物质界下方还有一个巨大的空洞世界，里面栖息着上古元素生物

### 3.2 暗影议会

暗影议会并非一味作恶，而是追求力量的另一种哲学——"以暗破晓，以影塑形"。

#### 暗影之手（暗杀与诅咒）
- **首领**：暗影领主（大法师巅峰，物质界最大威胁）
- **核心魔法**：《暗影步》《死亡诅咒》《灵魂收割》
- **领地**：暗影领域（被污染的魔脉）
- **当前目标**：腐蚀真理之塔，将暗影界与物质界融合
- **与其他势力关系**：与所有光明势力**死敌**
- **特色**：暗杀与诅咒能力天下第一，正面战斗偏弱
- **支线扩展点**：暗影之手内部有"温和派"（主张暗影也是元素之一，应被接纳）和"极端派"（主张吞噬光明），玩家可选择支持一派

#### 亡灵殿堂（死灵魔法）
- **殿主**：巫妖王（大法师中期，实为活人转亡灵）
- **核心魔法**：《亡灵天灾》《灵魂囚禁》《死亡凋零》
- **领地**：亡者之地（连接暗影界的裂缝）
- **当前目标**：建立亡灵国度，让所有生命"永生"
- **与其他势力关系**：与暗影之手**盟友**
- **特色**：操控亡灵作战，数量优势
- **支线扩展点**：巫妖王其实是一位千年前的大法师，他转亡灵是为了等待爱人的转世——他并非邪恶，只是一个不愿放手的人

#### 混沌教团（禁忌魔法）
- **教主**：混沌魔女（大法师初期，亦正亦邪）
- **核心魔法**：《混沌爆发》《元素逆转》《因果扭曲》
- **领地**：扭曲空间（小型魔脉，但空间不稳定）
- **当前目标**：打破魔法法则的束缚，让所有人都能使用魔法
- **与其他势力关系**：与所有势力**表面敌对，暗中交易**
- **特色**：混沌魔法威力巨大但完全不可控
- **支线扩展点**：混沌教团已渗透多个学院，玩家可发现内奸名单并选择公开或利用

### 3.3 德鲁伊联盟

德鲁伊并非人类组织，而是由精灵、树人、兽人等自然生灵组成的生态守护者联盟。

- **盟主**：世界树之灵（年龄未知，实力相当于大法师巅峰）
- **核心诉求**：维护自然平衡，反对过度使用魔法破坏生态
- **与人类关系**：与大地学院**中立偏友好**，与烈焰学院**紧张**（火系魔法常引发森林火灾），与暗影议会**死敌**
- **内部问题**：年轻一代德鲁伊主张"人类也应该被纳入自然循环"，老一辈主张"人类是破坏者，应该被驱逐"
- **支线扩展点**：玩家若为德鲁伊身份，将面临"站在自然一边"还是"融入人类社会"的抉择

### 3.4 元素神殿

- **大祭司**：元素圣女（大法师后期）
- **理念**：七元素皆为神圣，主张元素平衡
- **特殊能力**：元素调和——可以平息失控的元素魔法
- **支线扩展点**：神殿有一件至宝"元素之心"，据说可以暂时让一个人拥有所有元素的亲和力，各方势力都想获取

### 3.5 法师公会

- **性质**：松散的组织，没有统一领袖，只有"公会议会"
- **构成**：被学院抛弃的学徒、野法师、落魄贵族后裔
- **生存状态**：资源匮乏，常被学院和暗影议会欺压
- **支线扩展点**：法师公会正在酝酿一场"法师起义"，要打破学院对知识的垄断

## 4. 核心人物谱系

### 4.1 精灵公主艾拉（ romance / 潜在伴侣 ）

- **真实姓名**：艾拉瑞尔·月光
- **身份**：精灵王国的公主，德鲁伊联盟的外交使者
- **年龄**：外表十八许，实际年龄三百岁（高级法师）
- **性格**：优雅知性，热爱自然，对人类世界充满好奇但也有警惕
- **背景故事**：自幼被教导人类是自然的破坏者。一次偶然的机会，她看到一个人类法师用自己的魔力治愈了一片被污染的森林，这让她开始重新审视人类。
- **与玩家关系**：
  - 初遇：玩家在某次事件中保护了一片森林，她前来道谢
  - 发展：多次相遇后，她发现玩家不图她身份地位，单纯待她如普通人
  - 高潮：德鲁伊联盟与人类爆发冲突，她被迫在"种族责任"和"个人感情"之间做选择
  - 结局分支：结为伴侣 / 反目成仇 / 她为大义牺牲
- **秘密**：她其实是半精灵——她的母亲是一位人类法师。如果暴露，她将被剥夺公主身份
- **相关支线**：「人精之恋」「半精灵之秘」「自然与文明的抉择」

### 4.2 大法师梅林（ mentor / 潜在师父 ）

- **真实姓名**：梅林·安布罗修斯
- **身份**：前风暴学院教授，后因理念不合离开
- **年龄**：外表七十许，实际年龄一千五百岁（大法师后期）
- **性格**：睿智幽默，爱才如命，但偶尔会露出深不可测的一面
- **背景故事**：五百年前因反对学院的知识垄断政策而被逐出风暴学院。此后游历大陆，收养了多名孤儿为徒，但徒弟多死于各种"意外"。
- **与玩家关系**：
  - 初遇：玩家在幼年遭遇危险时被他救下
  - 发展：他看出玩家资质，主动传授基础魔法
  - 高潮：玩家发现他的徒弟"意外死亡"背后有惊天阴谋
  - 结局分支：正式拜师 / 发现他是真理之塔卧底 / 发现他被暗影议会控制
- **秘密**：他离开风暴学院的真正原因是发现了"预言之子预言"，一直在暗中寻找预言之子并保护之
- **相关支线**：「传承之路」「师父的秘密」「学院黑幕」

### 4.3 暗影领主（ rival / 终极敌人 ）

- **真实姓名**：马勒基斯（原名卢修斯·晨光）
- **身份**：暗影议会首领，暗影界在物质界的代理人
- **年龄**：三千岁（大法师巅峰，半只脚踏入贤者）
- **性格**：冷酷无情，为达目的不择手段，但对自己唯一的妹妹（已死）有过真情
- **背景故事**：本是光明神殿的圣子，因妹妹被光明神殿当作"暗影间谍"处决而堕入暗影。发誓要毁灭光明，重建一个"没有虚伪正义"的世界。
- **与玩家关系**：
  - 初遇：玩家在某次事件中破坏了他的暗影腐蚀计划
  - 发展：他发现玩家资质非凡，试图拉拢入暗影议会
  - 高潮：暗影战争最终决战
  - 结局分支：击败他 / 被他同化 / 理解他的痛苦后帮他解脱
- **秘密**：他其实是千年前那位被处决的女子的转世——也就是说，他堕入暗影的根本原因是"爱人为我而死，我却无法保护她"，这种痛苦持续了三千年
- **相关支线**：「暗影邀请」「血海深仇」「最后的救赎」

### 4.4 古龙长老（ mentor / 亦师亦敌 ）

- **真实姓名**：阿兹达哈卡（龙族与泰坦混血）
- **身份**：龙族最高议会成员
- **年龄**：一万岁（实力深不可测）
- **性格**：智慧深沉，重视承诺，有王者之风
- **背景故事**：父亲是龙族龙王，母亲是泰坦巨人，因混血身份在龙族内部备受歧视。靠实力一步步成为龙族最高议会成员。
- **与玩家关系**：
  - 初遇：玩家误入龙族领地，他本欲杀之，但发现玩家身怀龙族血脉
  - 发展：他邀请玩家参加"龙族试炼"，考验玩家的实力与品格
  - 高潮：人龙冲突爆发，他被迫在"龙族利益"和"对玩家的个人感情"之间抉择
  - 结局分支：成为龙族盟友 / 被他视为叛徒追杀 / 帮助他实现人龙和平
- **秘密**：他其实私下与人类某位高层有秘密协议，试图和平解决人龙矛盾，但双方都有激进派反对
- **相关支线**：「龙族试炼」「人龙和平」「混血的身份」

### 4.5 妖精女王（ romance / 调皮伴侣 ）

- **真实姓名**：汀克贝尔（自称，真名无人知晓）
- **身份**：妖精族女王
- **年龄**：外表十二岁，实际年龄未知
- **性格**：调皮可爱，喜欢恶作剧，但关键时刻极其可靠
- **背景故事**：妖精族是元素位面的原生种族，她可以随意穿梭于物质界和元素位面之间。她成为女王纯属意外——上一任女王在恶作剧时不小心把自己变成了青蛙，她作为王位的第七顺位继承人被迫登基。
- **与玩家关系**：
  - 初遇：玩家在某次召唤仪式中意外把她召唤了出来，她不但没有生气，反而觉得很有趣
  - 发展：她经常不打招呼就出现在玩家身边，有时是帮忙，有时只是恶作剧
  - 高潮：元素位面与物质界的通道即将崩溃，她必须在"拯救自己的世界"和"留在物质界"之间做选择
  - 结局分支：成为伴侣 / 回到元素位面 / 找到两全其美的方法
- **秘密**：她其实掌握着"元素契约"的最高权限——可以与任何生物签订元素共享契约，这意味着她可以让一个没有魔法天赋的人瞬间拥有元素之力
- **相关支线**：「妖精的恶作剧」「元素契约」「位面裂缝」

### 4.6 魔女莉莉丝（ rival / 亦敌亦友 ）

- **真实姓名**：莉莉丝·冯·海辛
- **身份**：混沌教团首领，前烈焰学院教授
- **年龄**：五百岁（大法师初期）
- **性格**：疯狂与理智并存，时而妩媚动人，时而冷酷无情
- **背景故事**：曾是烈焰学院最年轻的天才教授，因研究"禁忌魔法"被学院开除。她并非追求力量，而是想证明"魔法法则可以被打破"。
- **与玩家关系**：
  - 初遇：玩家在某次事件中急需魔法救命，她提出"以物易物"或"欠下人情"
  - 发展：她发现玩家有特殊的魔法天赋，邀请加入混沌教团
  - 高潮：她的实验即将成功——一种可以让魔法绝缘者也能使用魔法的药剂。但代价是实验需要一个活人作为媒介。
  - 结局分支：阻止她的实验 / 参与实验获得力量 / 发现她其实已经有不牺牲活人的替代方案，只是在测试人性
- **秘密**：她其实是被冤枉的——当年她研究的并非禁忌魔法，而是"替代能源魔法"，可以不消耗生命力施法。但学院高层害怕这种技术会颠覆现有的魔法经济体系，所以陷害了她。
- **相关支线**：「混沌入门」「学院黑市」「魔法革命」

## 7. 社会结构与文化习俗

### 7.1 法师社会阶层

```
凡人 → 魔法学徒 → 初级法师 → 中级法师 → 高级法师 → 大法师 → 贤者
```

- **凡人**：没有魔法天赋或魔法绝缘，无法施法。但在魔法大陆中占绝大多数（约99%）。
- **魔法学徒**：学院最底层，负责杂务，只有基础咒语。
- **初级法师**：正式学生，有独立实验室，可学习中级魔法。
- **中级法师**：教授亲传学生，学习高级魔法，是未来的大法师人选。
- **高级法师**：大法师以下最强，负责学院事务和教导学生。
- **大法师**：触摸魔法法则，可以创造原创魔法。
- **贤者**：传说中的境界，据说可以修改魔法法则本身。

> **社会现实**：阶层固化严重。平民学徒若无奇遇，几乎不可能进入学院；贵族天才若犯错，也可能被降为学徒甚至逐出学院。

### 7.2 婚姻制度

- **魔法契约**：法师的"婚姻"，需以魔力为引立下魔法誓言，违约会遭魔力反噬。
- **元素婚礼**：结为伴侣的仪式，每百年举行一次大型元素婚礼（由元素神殿主办）。
- **元素共享契约**：两位法师可以共享元素亲和力，可加速修炼速度，但需双方信任。
- **凡人恋禁忌**：法师与凡人相恋被视为"堕落"，因为凡人寿命太短，法师会因此魔力受损。

### 7.3 经济体系

- **魔晶**：通用货币，按品级分为下品、中品、上品、极品。1极品 = 100上品 = 10000中品 = 1000000下品。
- **学院贡献点**：学院内部货币，完成任务获得，可兑换魔法书、魔药、实验室。
- **黑市交易**：不受学院认可的地下交易，可买到禁忌物品（如暗影魔法书、活人实验材料）。

### 7.4 禁忌与规矩

| 禁忌行为 | 学院惩罚 | 暗影议会态度 |
|----------|----------|-------------|
| 弑师 | 废除魔力，逐出学院 | 无所谓，甚至鼓励 |
| 灵魂吞噬 | 灵魂焚烧 | 常见手段 |
| 叛院 | 追杀令，不死不休 | 欢迎加入 |
| 用凡人做实验 | 学院公敌 | 混沌教团日常 |
| 研究禁忌魔法 | 废除魔力 | 核心魔法 |
| 法师与凡人恋 | 降为学徒 | 无所谓 |

## 9. 支线故事种子（预留扩展）

以下种子可直接用于后续支线设计：

### 种子1：贤者之石
- **触发条件**：玩家为炼金术士身份，中级法师以上，进入真理之塔
- **故事梗概**：真理之塔地下实验室中有一处上古炼金工坊，内有贤者之石的炼制方法。但炼制需要一种早已灭绝的材料——凤凰之泪
- **可能结局**：成功炼制贤者之石 / 发现贤者之石其实是骗局 / 发现炼制贤者之石需要牺牲一位大法师的灵魂
- **相关势力/NPC**：烈焰学院、莉莉丝、大法师梅林

### 种子2：元素婚礼
- **触发条件**：玩家与某位NPC好感度达到亲密，年龄20岁以上
- **故事梗概**：每百年一次的元素婚礼即将举行，玩家可与心仪NPC结为伴侣。但婚礼上会有人捣乱——暗影议会试图在婚礼上释放"元素紊乱"咒语
- **可能结局**：顺利结为伴侣 / 婚礼被破坏 / 将计就计反制暗影议会
- **相关势力/NPC**：元素神殿、妖精女王、暗影领主

### 种子3：学院黑幕
- **触发条件**：玩家在学院区域发现异常，intelligence ≥ 60
- **故事梗概**：某个看似正常的学院实验其实是用学徒做活体实验。玩家选择举报（可能被学院打压）或暗中解救（难度极高）
- **可能结局**：揭露黑幕获得正义声望 / 解救受害者获得隐藏功德 / 被学院反咬一口
- **相关势力/NPC**：暗影议会、莉莉丝、大法师梅林

### 种子4：预言之子的真相
- **触发条件**：玩家 luck ≥ 100，且已完成多个隐藏事件
- **故事梗概**：玩家发现自己可能就是预言中的"预言之子"——出生时的魔法异象、逢凶化吉的运气、被各方势力关注，都不是偶然。但成为预言之子意味着被命运束缚，失去自由意志
- **可能结局**：接受天命成为命运代言人 / 反抗命运追求自由 / 假死脱身隐姓埋名
- **相关势力/NPC**：真理之塔、所有势力

### 种子5：贤者真相
- **触发条件**：玩家 reach 大法师期，且与梅林有过深入交流
- **故事梗概**：梅林透露"贤者已死"的真相——不是无法成为贤者，而是星界有人在真理之塔顶层设下封印，阻止凡人触及真理。这个"人"可能是星界的某位存在，目的是独占真理
- **可能结局**：强行突破封印（九死一生）/ 找到替代方法 / 放弃成为贤者留在物质界守护
- **相关势力/NPC**：法师公会、所有大法师

### 种子6：人龙和平
- **触发条件**：玩家为龙血混血或德鲁伊身份，与古龙长老好感度达到友好
- **故事梗概**：古龙长老私下与人类某位高层达成和平协议，但双方激进派都不满意。玩家作为中间人，需要在各种阴谋和暗杀中保护协议
- **可能结局**：促成和平 / 协议破裂人龙开战 / 发现协议本身是个陷阱
- **相关势力/NPC**：龙族、大地学院、所有学院

### 种子7：元素之心的秘密
- **触发条件**：玩家与元素神殿有过交易，intelligence ≥ 80
- **故事梗概**：元素神殿正在秘密研究一件"元素之心"，但制造需要活人作为元素容器。大祭司声称这是"必要的牺牲"，为了阻止元素位面崩溃
- **可能结局**：阻止制造 / 参与制造获得元素之心 / 发现大祭司其实是被星界某人操控的傀儡
- **相关势力/NPC**：元素神殿、德鲁伊联盟（反对）、暗影议会（想抢）

### 种子8：师父的秘密
- **触发条件**：玩家拜梅林为师，且已完成大部分主线
- **故事梗概**：玩家发现梅林的徒弟们并非"意外死亡"——他们都是被真理之塔暗杀的，因为真理之塔推演出他们都是"潜在的预言之子"。梅林保护玩家的同时，也在利用玩家引出真理之塔
- **可能结局**：理解师父的苦衷 / 与师父决裂 / 联手摧毁真理之塔
- **相关势力/NPC**：梅林、真理之塔

### 种子9：妖精的抉择
- **触发条件**：玩家与妖精女王好感度达到爱慕
- **故事梗概**：元素位面与物质界的通道即将崩溃，妖精女王面临"拯救元素位面"或"留在物质界"的抉择。拯救元素位面需要她献出自己的元素之心，永远失去自由；留在物质界则元素位面崩溃，无数元素生物死亡
- **可能结局**：帮助她拯救元素位面 / 帮助她留在物质界 / 找到第三条路（用玩家的生命力为她续命）
- **相关势力/NPC**：妖精女王、元素神殿（主张拯救）、暗影议会（想利用崩溃）

### 种子10：法师起义
- **触发条件**：玩家为平民学徒身份，charisma ≥ 100
- **故事梗概**：法师公会酝酿起义，要求学院开放知识、废除阶层歧视。玩家可选择领导起义、镇压起义、或从中调解
- **可能结局**：起义成功建立新秩序 / 起义失败法师被屠杀 / 达成妥协建立"法师议会"
- **相关势力/NPC**：法师公会、所有学院

### 种子11：暗影领主的救赎
- **触发条件**：玩家与暗影领主有过深入对话（非战斗），intelligence ≥ 120
- **故事梗概**：玩家发现暗影领主堕入暗影的真正原因——三千年前，他的妹妹被光明神殿当作"暗影间谍"处决，而执行处决的正是如今某位学院高层。暗影领主不是要毁灭世界，而是要"让光明承认他们的错误"
- **可能结局**：帮他复仇 / 说服他放下仇恨 / 发现他的妹妹其实没死（被秘密囚禁）
- **相关势力/NPC**：暗影领主、所有学院

### 种子12：元素紊乱
- **触发条件**：玩家进入元素裂隙，且已沾染暗影魔力
- **故事梗概**：元素神殿至宝"元素之心"可以净化暗影魔力，但进入元素之心需要"放下执念"——玩家最执着的愿望会被元素幻境考验
- **可能结局**：净化成功 / 执念太深被元素反噬 / 发现元素之心本身才是最大的暗影源（它靠吸收执念成长）
- **相关势力/NPC**：元素神殿、暗影领主（想毁塔）、亡灵殿堂（想盗塔）

### 种子13：龙族血脉的觉醒
- **触发条件**：玩家为龙血混血，age ≥ 20
- **故事梗概**：玩家体内的龙族血脉开始觉醒，但觉醒过程会丧失理智。古龙长老邀请玩家回龙族接受"龙族试炼"，但学院认为玩家"堕入龙道"要追杀之
- **可能结局**：成功觉醒获得龙族力量 / 觉醒失败沦为野兽 / 找到压制血脉的方法保持人形
- **相关势力/NPC**：龙族、烈焰学院、古龙长老

### 种子14：真理之塔的阴谋
- **触发条件**：玩家与真理之塔有过接触，且luck ≥ 150
- **故事梗概**：真理之塔推演出"预言之子"的身份后，并非保护而是猎杀——因为真理之塔第一代塔主就是上一个"预言之子"，他知道成为预言之子的代价是被命运同化、失去自我。真理之塔猎杀预言之子，其实是在"拯救"他们
- **可能结局**：理解真理之塔 / 摧毁真理之塔 / 发现真理之塔塔主已经被命运反噬
- **相关势力/NPC**：真理之塔、梅林

### 种子15：最后的真理
- **触发条件**：玩家 reach 贤者预备阶段
- **故事梗概**：玩家准备通过真理之塔顶层试炼成为贤者，但发现试炼被人动了手脚——有人在试炼中加入了"灭魂咒"，这是星界黑手的手段，目的是阻止凡人成为贤者。玩家需要在试炼的同时对抗星界黑手
- **可能结局**：成功成为贤者并反杀星界黑手 / 试炼失败但摧毁了封印 / 放弃成为贤者留在物质界成为守护神
- **相关势力/NPC**：所有势力、星界（隐藏）
